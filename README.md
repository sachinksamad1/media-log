# Media Log

A simple media log application built with Node.js, Express, and Firebase.

## Features

- Add, update, and delete media items
- Search for media items by name
- Filter media items by type
- Sort media items by name and date added
- Pagination for large datasets
- Responsive design

## Technologies Used

- Node.js
- Express
- Firebase
- TypeScript
- Tailwind CSS

## Setup

1. Clone the repository:
```bash
$ git clone https://github.com/your-username/media-log.git
$ cd media-log
```

2. Install dependencies:
```bash
$ npm install
```

3. Set up Firebase:
- Create a Firebase project
- Add the Firebase configuration to the `config/firebase.js` file

4. Run the application:
```bash
$ npm run dev
```

The application will be available at `http://localhost:3000`.
